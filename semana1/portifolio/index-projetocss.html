Loja Mahin.Kombucha

DESCRIÇÃO:

Projeto realizado para a marca Mahin.Kombuchas, loja que vende bebida kombucha em lata com uma identidade visual marcada pelo estilo saudável e natural.


O objetivo desse projeto com CSS era estilizar o site de acordo com o stilo da marca, trazendo uma maior identidade visual para a página web. Foram usadas cores que trabalham em hamornia com os produtos.
Tecnologias Utilizadas:
  • HTML
  • CSS
